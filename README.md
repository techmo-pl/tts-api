# Techmo TTS Service API

Techmo TTS Service API is defined in `proto/techmo_tts.proto` file.
Consult comments in the proto file for description of the supported GRPC calls.

**NOTE:** this repository includes API for Techmo TTS DNN Service version 3.0 and higher.
Proto files for older versions of the service can be found at: https://github.com/techmo-pl/tts-client/blob/master/proto/techmo_tts.proto
